 # Project Overview

A real-time technical interview and coding platform designed to simulate real-world interviews with live collaboration, secure code execution, and automated evaluation.

 # Key Features

Secure authentication and private 1-on-1 interview rooms powered by Clerk

Live video interviews with screen sharing, recording, mic/camera controls, and real-time chat using Stream

VSCode-like coding environment with isolated code execution and auto success/failure feedback based on test cases

Interactive dashboard with live stats, solo practice problems, and instant visual feedback

Scalable backend with Node.js & Express, async background jobs via Inngest, and efficient caching using TanStack Query

# Tech Stack

Frontend: React, TanStack Query

Backend: Node.js, Express, REST APIs

Auth: Clerk

Video & Chat: Stream Video, Stream Chat

Async Jobs: Inngest
